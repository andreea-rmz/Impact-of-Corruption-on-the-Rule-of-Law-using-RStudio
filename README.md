# Impact-of-Corruption-on-the-Rule-of-Law-using-RStudio

Empirical analysis of the World Justice Project Rule of Law Index (2022) for 140 countries. Includes a multiple linear regression model, hypothesis testing, correlation and graphical analysis, and VIF diagnostics to assess how institutional factors—especially absence of corruption—affect overall rule-of-law performance.

*********************************************************************************************************************************************************************************************************************************************************
PROJECT DESCRIPTION
*********************************************************************************************************************************************************************************************************************************************************
This project analyzes the institutional determinants of the Rule of Law using the World Justice Project (WJP) Rule of Law Index 2022 for 140 countries. A multivariate linear regression model is estimated to evaluate the marginal effect of eight institutional components—particularly absence of corruption—on overall rule-of-law performance. The analysis includes hypothesis testing, correlation matrices, scatterplots and multicollinearity diagnostics using Variance Inflation Factors (VIF).

Results show that absence of corruption has a positive and statistically significant effect on the Rule of Law score, confirming that lower corruption levels strengthen institutional quality. Regulatory enforcement, criminal justice and civil justice display the strongest correlations with the overall Index. VIF results indicate high collinearity among institutional components, which is consistent with the interconnected nature of the rule-of-law framework.

*********************************************************************************************************************************************************************************************************************************************************
DESCRIPCIÓN DEL PROYECTO
*********************************************************************************************************************************************************************************************************************************************************

Este proyecto analiza los determinantes institucionales del Estado de Derecho utilizando el Índice de Estado de Derecho del World Justice Project (WJP) 2022 para 140 países. Se estima un modelo de regresión lineal multivariado para evaluar el efecto marginal de ocho componentes institucionales —especialmente la ausencia de corrupción— sobre el puntaje general del Estado de Derecho. El análisis incluye pruebas de hipótesis, matrices de correlación, gráficos de dispersión y diagnósticos de multicolinealidad mediante el Factor de Inflación de la Varianza (VIF).

Los resultados muestran que la ausencia de corrupción tiene un efecto positivo y estadísticamente significativo sobre el puntaje del Estado de Derecho, lo que confirma que menores niveles de corrupción fortalecen la calidad institucional. Los factores de cumplimiento regulatorio, justicia penal y justicia civil presentan las correlaciones más fuertes con el índice general. Los valores elevados de VIF indican alta colinealidad entre los componentes institucionales, consistente con la naturaleza interrelacionada del marco del Estado de Derecho.
